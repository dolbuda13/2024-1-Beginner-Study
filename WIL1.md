1주차
========================
git을 사용하는 이유
-----------------
>1. 코드 수정 관리
>2. 파일 관리
>3. 협업
>4. 버전 관리

git의 영역
--------------------
>1. working directory
>2. staging area
>3. repository

git으로 파일 광리하기
-------------------
>1. 저장소 생성(git init)
>2. 관리 대상 등록(git add)
>3. 파일 수정 후 로컬 저장소로 옮기기 (git commit)
